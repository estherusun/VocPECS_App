# VocPECS 🌿
**Vocational training for People with Disabilities (PWD) through digitized PECS and Kanban systems.**

## 📌 Project Overview
VocPECS is an autism-friendly mobile application designed to digitize agricultural training workflows (Fertigation & Hydroponics). It translates complex tasks into linear, visual steps to reduce cognitive overload and encourage independent learning.

## 🚀 Key Features
- **Visual Login:** Netflix-style profile system for personalized user data.
- **Creation Hub:** Scan physical tools via camera/AI to create custom PECS cards.
- **5-Column Kanban:** A structured "Structured Teaching" flow (Tools -> To Do -> Doing -> Done -> Reward).
- **Instructional Scaffolding:** Distraction-free "Focus Mode" popups for every step.
- **Token Economy:** Visual treat rewards (cake, candy, cookie) upon task completion.

## 🛠️ Built With
- **Framework:** Flutter (Dart)
- **State Management:** Provider
- **AI Engine:** Google ML Kit (Object Labeling)
- **Local Database:** SharedPreferences
- **UI Principles:** TEACCH Model & Universal Design for Learning (UDL)

## 📁 Folder Structure
- `lib/models/`: Data blueprints (PECS Items, Profiles).
- `lib/providers/`: The app's logic and state management.
- `lib/screens/`: Minimalist, cream-themed user interfaces.
- `lib/data/`: Modularized agricultural module content.
- `assets/images/`: Visual assets for Edamame modules.
